#  Weather Dashboard

A minimal, modern, and responsive Weather Dashboard built using vanilla web technologies. This application fetches real-time weather data from the OpenWeatherMap API and dynamically updates the user interface and background styling based on the current weather conditions of the searched city.

 **Live Demo:** [View Live Site](https://chamudierangi.github.io/Weather-Dashboard/)



##  Features

* **Real-Time Data:** Fetches up-to-date temperature, weather condition descriptions, and dynamic weather icons.
* **Dynamic Backgrounds:** Smoothly transitions the page background image based on the current weather state (e.g., Sunny, Cloudy, Rainy, Misty).
* **Glassmorphic UI:** Features a modern, sleek glassmorphism card design for maximum readability and visual appeal.
* **Seamless UX:** Supports searching via both the "Search" button and the "Enter" key, with built-in error handling for invalid city names.



## Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox, Glassmorphism, CSS Transitions)
* **Scripting:** JavaScript (ES6+, Async/Await, Fetch API, DOM Manipulation)
* **API Source:** [OpenWeatherMap API](https://openweathermap.org/)
